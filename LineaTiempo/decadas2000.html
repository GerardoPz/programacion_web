Aplicaciones Web Interactivas

La revolución de las verdaderas Aplicaciones Web. Pasamos de consumir contenido a generarlo e interactuar con él (redes sociales, foros, blogs).

  • 2004: Se populariza el concepto de Web 2.0. Nace Facebook y la interacción social en línea.
  • 2005: Google lanza Google Maps y populariza AJAX (Asynchronous JavaScript and XML). Esto permitió que las páginas web se actualizaran en tiempo real sin tener que recargar toda la ventana.
  • 2008: El lanzamiento de HTML5 y Google Chrome acelera enormemente el rendimiento de las aplicaciones web en el navegador.
