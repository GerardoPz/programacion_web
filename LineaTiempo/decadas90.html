Navegadores y Páginas Estáticas

Esta es la década donde la web se vuelve pública. Nace la Web 1.0, caracterizada por sitios de "solo lectura" donde los usuarios consumían información sin interactuar.

  • 1991: Se publica la primera página web de la historia (creada con HTML).
  • 1993: Se lanza Mosaic, el primer navegador web con interfaz gráfica, popularizando Internet entre el público general.
  • 1995: Brendan Eich crea JavaScript en solo 10 días, introduciendo la posibilidad de hacer las páginas ligeramente interactivas.
